# Query Busters

A SQL-themed browser game built with React + Vite. SQL code scrolls up the screen — shoot it down before it reaches the top.

---

## Running Locally

### Prerequisites

You'll need **Node.js** installed. If you're not sure whether you have it, open a terminal and run:

```
node -v
```

If you see a version number, you're good. If not, download and install it from [nodejs.org](https://nodejs.org) (the "LTS" version is recommended).

---

### Setup

**1. Clone the repository**

```
git clone <repo-url>
cd query-busters
```

**2. Install dependencies**

```
npm install
```

**3. Start the game**

```
npm run dev
```

**4. Open your browser and go to:**

```
http://localhost:5173
```

---

## How to Play

| Control | Action |
|---|---|
| `Hold Space` | Start game / Fire |
| `← →` Arrow Keys | Move left / right |
| `Enter` | Restart after game over |

Shoot down the SQL before it reaches the top of the screen. The further the query gets, the higher your score when it's game over.
